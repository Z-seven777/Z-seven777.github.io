宇宙的量子之舞：从星云到黑洞

2023-10-05

🌌 星云：宇宙的摇篮

在猎户座大星云的深处，氢原子在引力牵引下缓慢坍缩，孕育出新生恒星。这些恒星托儿所的温度高达 10,000°C，却因稀薄的气体密度显得异常宁静。当第一缕核聚变的光芒刺破尘埃时，宇宙便完成了一次无声的加冕。

# 示例代码：计算恒星形成的最小质量（Jean's Mass）
def jeans_mass(temperature, density):
    G = 6.674e-11  # 引力常数
    k = 1.38e-23    # 玻尔兹曼常数
    mu = 2.3e-27    # 平均分子量（kg）
    return (k * temperature / (G * mu**2 * density))**(3/2) * (5 / (2 * 3.14))**0.5
print(f"临界质量: {jeans_mass(10000, 1e-15):.2f} 太阳质量")
